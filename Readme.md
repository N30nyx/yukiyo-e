# yukiyo-e

### Requirements:
Install python from https://python.org
### Installation:
Windows:
open command prompt:
```
git clone https://github.com/N30nyx/yukiyo-e
install
python3 main.py
```

Linux/Mac:
open terminal:
```
git clone https://github.com/N30nyx/yukiyo-e
bash install.sh
python3 main.py
```
